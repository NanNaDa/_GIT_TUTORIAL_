# SQLite - Swift & SwiftUI

## Install the SQLite library in your project

1. pod initialization
```
$ pod init 
```

2. add library in Podfile
```
# Pods for SwiftUI_CRUD
	pod 'SQLite.swift', '~> 0.12.0'
```

3. pod upate
```
$ pod update
```

4. start project in Xcode
